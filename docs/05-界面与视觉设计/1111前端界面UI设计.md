Props配置：

action: 上传地址（默认'/api/file/upload'）

limit: 最大上传数量（默认5）

maxSize: 文件大小限制MB（默认10）

autoUpload: 是否自动上传（默认true）
buttonText: 按钮文本（默认'选择文件'）

文本模式: 显示文件名列表
