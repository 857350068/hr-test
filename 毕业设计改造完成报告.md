## 二、核心功能实现

### 1. Hive数据源集成 ✅

**修改文件**：
- `backend/pom.xml` - 添加Hadoop和Hive依赖
- `backend/src/main/resources/application.yml` - 配置多数据源

**新增文件**：
- `MySQLDataSourceConfig.java` - MySQL数据源配置
- `HiveDataSourceConfig.java` - Hive数据源配置

**关键配置**：
```yaml
spring:
  datasource:
    mysql:
      driver-class-name: com.mysql.cj.jdbc.Driver
      url: jdbc:mysql://localhost:3306/hr_datacenter
    hive:
      driver-class-name: org.apache.hive.jdbc.HiveDriver
      url: jdbc:hive2://localhost:10000/hr_datacenter_dw
```

### 2. 数据同步机制 ✅

**新增文件**：
- `DataSyncService.java` - 数据同步服务
- `ScheduleConfig.java` - 定时任务配置

**同步策略**：
- 维度表：每日凌晨2点全量同步
- 事实表：每小时增量同步
- 聚合表：每日凌晨3点重新计算

**核心指标**：

#### 3.3 薪酬福利分析

## 五、部署说明

### 1. 环境要求

**必需环境**：
- JDK 1.8+
- MySQL 8.0+
- Hadoop 3.2.4
- Hive 3.1.3
- Node.js 16+

### 2. Hive环境配置

**启动Hive服务**：
```bash
# 启动Hive Thrift服务
hive --service hiveserver2

# 验证Hive连接
beeline -u jdbc:hive2://localhost:10000
```

**初始化Hive数据仓库**：
```bash
# 执行Hive初始化脚本
hive -f database/hive/init.sql
```

### 3. 后端启动

```bash
# 进入后端目录
cd backend

# Maven编译打包
mvn clean package

# 运行项目
mvn spring-boot:run
```

### 4. 前端启动

```bash
# 进入前端目录
cd frontend

# 安装依赖
npm install

# 运行项目
npm run dev
```
