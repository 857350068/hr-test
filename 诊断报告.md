## 三、问题诊断

### 图表不显示的可能原因

#### 1. 数据库未初始化 ⚠️
**症状**：图表区域空白，无数据显示
**原因**：数据库表为空，查询返回空结果
**解决方案**：
```bash
# 执行数据库初始化脚本
mysql -u root -p < database/mysql/init.sql
```

#### 2. 后端服务未启动 ⚠️
**症状**：浏览器控制台显示网络错误
**原因**：后端服务未运行，前端无法调用API
**解决方案**：
```bash
# 启动后端服务
cd backend
mvn spring-boot:run
```

#### 3. 前端未构建或未启动 ⚠️
**症状**：页面无法访问或显示异常
**原因**：前端代码未编译或开发服务器未运行
**解决方案**：
```bash
# 启动前端开发服务器
cd frontend
npm install
npm run dev
```

#### 4. 跨域问题 ⚠️
**症状**：浏览器控制台显示CORS错误
**原因**：前后端端口不同，跨域配置问题
**解决方案**：检查 `CorsConfig.java` 配置

#### 5. 认证问题 ⚠️
**症状**：API返回401未授权错误
**原因**：未登录或Token过期
**解决方案**：先登录系统获取有效Token
