# Word 排版参数终检清单（提交前逐项打勾）

## 1. 页面与装订

- [ ] 纸张 A4
- [ ] 上边距 3.0 cm
- [ ] 下边距 2.5 cm
- [ ] 左边距 2.5 cm
- [ ] 右边距 2.0 cm
- [ ] 页脚 1.75 cm
- [ ] 封面与封底使用白色铜版纸（120g-150g）

## 2. 正文字体与段落（建议统一）

- [ ] 正文：宋体，小四，首行缩进 2 字符，两端对齐，固定行距 20 磅
- [ ] 一级标题（第一章）：黑体，小三，加粗，段前段后适中
- [ ] 二级标题（1.1）：黑体，四号，加粗
- [ ] 三级标题（1.1.1）：黑体，小四，加粗
- [ ] 中英文摘要标题：黑体，小三
- [ ] 关键词标签“关键词/Key Words”：黑体，小四

## 3. 目录

- [ ] 使用 Word 自动目录生成
- [ ] 目录层级到三级标题
- [ ] 目录与正文标题完全一致
- [ ] 页码正确

## 4. 参考文献（GB/T 7714）

- [ ] 文献数量不少于 20 篇
- [ ] 按 [1][2]…连续编号
- [ ] 中英文文献格式统一
- [ ] 每条文献采用悬挂缩进 2 字符
- [ ] 标点全角/半角风格统一

## 5. 学校要求内容完整性

- [ ] 英文题目与作者
- [ ] 任务书（签字）
- [ ] 开题报告
- [ ] 原创性声明与授权书
- [ ] 中文摘要（300-500字）
- [ ] 英文摘要（与中文一致）
- [ ] 关键词 3-5 个
- [ ] 正文（不少于 8000 字）
- [ ] 结论（单独章节）
- [ ] 参考文献（>=20）
- [ ] 致谢
- [ ] 附录

## 6. 附件材料（单独装订）

- [ ] 英文文献原文（>=3000词）
- [ ] 中文翻译（完整对应）
- [ ] 专业阅读书目 >=10 篇
- [ ] 每篇阅读书目 >=300 字
- [ ] 在正文或开题中可对应引用
