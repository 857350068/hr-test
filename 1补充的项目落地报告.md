MySQL→Hive 数据同步模块（核心）开题要求 “同步 MySQL 基础数据至 Hive”，基于 Spring Schedule 实现定时同步 + 数据清洗：
java
运行
@Service
public class DataSyncService {
    @Autowired
    private EmployeeMapper employeeMapper; // MySQL Mapper
    @Autowired
    private JdbcTemplate hiveJdbcTemplate; // Hive JDBC模板

    // 每天凌晨1点同步MySQL基础数据到Hive
    @Scheduled(cron = "0 0 1 * * ?")
    public void syncMysqlToHive() {
        // 1. 从MySQL查询员工基础数据
        List<Employee> empList = employeeMapper.selectAll();
        // 2. 数据清洗：处理空值、格式统一
        List<String> hiveSqlList = empList.stream()
                .filter(emp -> emp.getDept() != null) // 过滤空部门数据
                .map(emp -> String.format(
                        "INSERT INTO hr_hive.employee VALUES ('%s','%s','%s','%s')",
                        emp.getId(), emp.getName(), emp.getDept(), emp.getSalary()
                ))
                .collect(Collectors.toList());
        // 3. 批量插入Hive
        hiveSqlList.forEach(sql -> hiveJdbcTemplate.execute(sql));
    }
}
RBAC 权限控制模块（多角色适配）开题要求 “HR 管理员 / 部门负责人 / 管理层” 的权限隔离，基于 Spring Security 实现：
java
运行
@Configuration
@EnableWebSecurity
public class SecurityConfig {
    @Bean
    public SecurityFilterChain filterChain(HttpSecurity http) throws Exception {
        http.authorizeHttpRequests(auth -> auth
                // 后台管理员仅HR_ADMIN可访问
                .requestMatchers("/api/admin/**").hasRole("HR_ADMIN")
                // 部门数据仅DEPT_LEADER可访问
                .requestMatchers("/api/dept/**").hasRole("DEPT_LEADER")
                // 分析看板仅MANAGER可访问
                .requestMatchers("/api/dashboard/**").hasRole("MANAGER")
                // 公共接口（登录/注册）放行
                .requestMatchers("/api/auth/**").permitAll()
                .anyRequest().authenticated()
        ).formLogin().and().logout();
        return http.build();
    }
}
流失预警模型模块（简化版落地）开题要求 “员工流失预警模型、调整特征权重”，先实现基于阈值的基础模型（后期可扩展机器学习）：
java
运行
@Service
public class WarningModelService {
    @Autowired
    private JdbcTemplate hiveJdbcTemplate;

    // 计算员工流失预警分数（考勤<80 + 绩效<60 → 高风险）
    public List<EmpWarningDTO> calculateQuitWarning() {
        String hiveSql = "SELECT id,name,attendance,performance FROM hr_hive.emp_analysis";
        List<Map<String, Object>> dataList = hiveJdbcTemplate.queryForList(hiveSql);
        
        return dataList.stream()
                .map(item -> {
                    EmpWarningDTO dto = new EmpWarningDTO();
                    dto.setEmpId(item.get("id").toString());
                    dto.setEmpName(item.get("name").toString());
                    double attendance = Double.parseDouble(item.get("attendance").toString());
                    double performance = Double.parseDouble(item.get("performance").toString());
                    
                    // 预警规则（权重可从配置表读取，支持管理员调整）
                    int score = 0;
                    if (attendance < 80) score += 50; // 考勤权重50%
                    if (performance < 60) score += 50; // 绩效权重50%
                    
                    dto.setWarningScore(score);
                    dto.setWarningLevel(score >= 80 ? "高风险" : score >= 40 ? "中风险" : "低风险");
                    return dto;
                })
                .collect(Collectors.toList());
    }
}
