## 🚀 如何启动项目

### 1. 数据库初始化

```bash
# 登录MySQL
mysql -u root -p

# 执行初始化脚本
source hr-backend/src/main/resources/sql/init.sql
```

### 2. 启动后端

```bash
cd hr-backend
mvn clean package -DskipTests
java -jar target/hr-data-center-1.0.0.jar
```

### 3. 启动前端

```bash
cd hr-frontend
npm install
npm run dev
```

### 4. 访问系统

- **前端地址**: http://localhost:3000
- **后端地址**: http://localhost:8080/api

### 5. 测试账号

- 用户名: `admin` / 密码: `123456`

## 🔧 常见问题

### Q1: 如何测试考勤打卡功能?

A: 
1. 登录系统
2. 点击左侧菜单"考勤打卡"
3. 点击"上班打卡"按钮
4. 系统会自动记录打卡时间
5. 点击"下班打卡"按钮完成下班打卡

### Q2: 如何申请请假?

A:
1. 点击左侧菜单"请假管理"
2. 点击"申请请假"按钮
3. 填写请假信息(类型、时间、原因)
4. 提交申请
5. 等待审批

### Q3: 如何审批请假?

A:
1. 点击左侧菜单"请假管理"
2. 查看待审批列表(或点击"待审批"按钮)
3. 点击"审批"按钮
4. 选择审批结果(同意/拒绝)
5. 填写审批意见
6. 提交审批

### Q4: 数据库如何初始化?

A:
```bash
mysql -u root -p
source hr-backend/src/main/resources/sql/init.sql
```
