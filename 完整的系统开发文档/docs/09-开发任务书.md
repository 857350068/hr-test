## 一、项目概述

### 1.1 项目背景

人力资源数据中心系统是一个基于Spring Boot + Vue3构建的企业级HR数据分析平台，采用MySQL + Hive双数据库架构，支持组织效能分析、人才梯队建设、薪酬福利分析、绩效管理、员工流失预警、培训效果评估、人力成本优化、人才发展预测等八大核心功能模块。

### 1.2 项目目标

- 构建统一的人力资源数据中心
- 实现数据可视化和分析
- 提供智能预警和决策支持
- 提升人力资源管理效率

### 1.3 项目周期

**总周期**: 7周

**开发环境**: Windows 11

**生产环境**: CentOS 7 虚拟机 (IP: 192.168.168.100)
