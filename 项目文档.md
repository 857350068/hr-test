## 项目简介

人力资源数据中心系统是一个基于大数据技术的人力资源管理平台，整合了员工管理、考勤管理、绩效管理、薪酬管理、培训管理等核心功能模块，通过数据分析和可视化，帮助企业实现人力资源管理的数字化、智能化。

### 主要目标

1. **数据整合**: 整合分散的人力资源数据，建立统一的数据中心
2. **数据分析**: 提供多维度、多层次的数据分析能力
3. **决策支持**: 为人力资源决策提供数据支持和智能建议
4. **风险预警**: 实时监控人力资源风险，及时预警
5. **效率提升**: 提升人力资源管理效率，降低管理成本

## 部署说明

### 生产环境部署

#### 后端部署

1. **打包项目**
```bash
cd backend
mvn clean package -DskipTests
```

2. **上传jar包到服务器**
```bash
scp target/hr-datacenter-1.0.0.jar user@server:/opt/app/
```

3. **创建启动脚本**
```bash
#!/bin/bash
nohup java -jar /opt/app/hr-datacenter-1.0.0.jar > /opt/app/logs/app.log 2>&1 &
```

#### 前端部署

1. **构建生产版本**
```bash
cd frontend
npm run build
```

2. **配置Nginx**
```nginx
server {
    listen 80;
    server_name your-domain.com;

    location / {
        root /usr/share/nginx/html;
        index index.html;
        try_files $uri $uri/ /index.html;
    }

    location /api {
        proxy_pass http://localhost:8080;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

### Docker部署

#### 后端Dockerfile

```dockerfile
FROM openjdk:8-jdk-alpine
VOLUME /tmp
COPY target/hr-datacenter-1.0.0.jar app.jar
ENTRYPOINT ["java","-jar","/app.jar"]
```

#### 前端Dockerfile

```dockerfile
FROM node:16-alpine as build
WORKDIR /app
COPY package*.json ./
RUN npm install
COPY . .
RUN npm run build

FROM nginx:alpine
COPY --from=build /app/dist /usr/share/nginx/html
COPY nginx.conf /etc/nginx/conf.d/default.conf
```

## 常见问题

### 1. 数据库连接失败

**问题**: 无法连接到MySQL数据库

**解决方案**:
- 检查数据库服务是否启动
- 验证数据库配置信息是否正确
- 确认数据库用户权限是否足够

### 2. 端口冲突

**问题**: 8080端口被占用

**解决方案**:
- 修改`backend/src/main/resources/application.yml`中的`server.port`
- 或者停止占用8080端口的服务

### 3. Lombok注解不生效

**问题**: Lombok注解编译时报错

**解决方案**:
- 确保IDE安装了Lombok插件
- 检查pom.xml中Lombok依赖是否正确
- 清理并重新编译项目：`mvn clean compile`

### 4. 跨域问题

**问题**: 前端请求后端接口时出现跨域错误

**解决方案**:
- 后端已配置CORS，检查配置是否正确
- 确认前端代理配置是否正确

### 5. Hive连接失败

**问题**: 无法连接到Hive数据库

**解决方案**:
- 确保Hadoop和Hive环境已正确配置
- 检查Hive服务是否启动
- 验证Hive连接配置信息

### 数据库表结构

#### 用户表 (sys_user)
- user_id: 用户ID（主键）
- username: 用户名（唯一）
- password: 密码（BCrypt加密）
- real_name: 真实姓名
- dept_id: 部门ID
- phone: 手机号码
- email: 邮箱
- status: 用户状态（0-禁用 1-启用）
- last_login_time: 最后登录时间
- create_time: 创建时间
- update_time: 更新时间
- deleted: 删除标记

#### 员工表 (employee)
- emp_id: 员工ID（主键）
- emp_no: 员工编号（唯一）
- emp_name: 员工姓名
- gender: 性别（0-女 1-男）
- birth_date: 出生日期
- id_card: 身份证号
- phone: 手机号码
- email: 邮箱
- department: 部门
- position: 职位
- salary: 薪资
- hire_date: 入职日期
- resign_date: 离职日期
- status: 员工状态（0-离职 1-在职 2-试用）
- education: 学历
- create_time: 创建时间
- update_time: 更新时间
- deleted: 删除标记

#### 考勤表 (attendance)
- attendance_id: 考勤ID（主键）
- emp_id: 员工ID
- attendance_date: 考勤日期
- checkin_time: 上班打卡时间
- checkout_time: 下班打卡时间
- status: 考勤状态（正常/迟到/早退/旷工）
- work_hours: 工作时长
- create_time: 创建时间
- update_time: 更新时间

#### 请假表 (leave)
- leave_id: 请假ID（主键）
- emp_id: 员工ID
- leave_type: 请假类型（事假/病假/年假等）
- start_date: 开始日期
- end_date: 结束日期
- days: 请假天数
- reason: 请假原因
- status: 审批状态（待审批/已批准/已拒绝）
- approver: 审批人
- approve_time: 审批时间
- create_time: 创建时间
- update_time: 更新时间

#### 绩效目标表 (performance_goal)
- goal_id: 目标ID（主键）
- emp_id: 员工ID
- goal_name: 目标名称
- goal_description: 目标描述
- target_value: 目标值
- actual_value: 实际值
- start_date: 开始日期
- end_date: 结束日期
- status: 目标状态（进行中/已完成）
- weight: 权重
- create_time: 创建时间
- update_time: 更新时间

#### 绩效评估表 (performance_evaluation)
- evaluation_id: 评估ID（主键）
- emp_id: 员工ID
- goal_id: 目标ID
- self_score: 自评分数
- manager_score: 上级评分
- total_score: 综合分数
- grade: 绩效等级（S/A/B/C/D）
- self_comment: 自评意见
- manager_comment: 上级评价
- interview_date: 面谈日期
- interview_summary: 面谈总结
- improvement_plan: 改进计划
- create_time: 创建时间
- update_time: 更新时间

#### 薪资发放表 (salary_payment)
- payment_id: 发放ID（主键）
- emp_id: 员工ID
- payment_date: 发放日期
- base_salary: 基本工资
- bonus: 奖金
- deduction: 扣款
- total_salary: 实发工资
- status: 发放状态（待发放/已发放）
- create_time: 创建时间
- update_time: 更新时间

#### 薪资调整表 (salary_adjustment)
- adjustment_id: 调整ID（主键）
- emp_id: 员工ID
- adjustment_type: 调整类型（调薪/晋升/降职）
- before_salary: 调整前薪资
- after_salary: 调整后薪资
- reason: 调整原因
- effective_date: 生效日期
- status: 审批状态（待审批/已批准/已拒绝）
- approver: 审批人
- approve_time: 审批时间
- create_time: 创建时间
- update_time: 更新时间

#### 培训课程表 (training_course)
- course_id: 课程ID（主键）
- course_name: 课程名称
- course_description: 课程描述
- instructor: 讲师
- start_date: 开始日期
- end_date: 结束日期
- max_participants: 最大人数
- current_participants: 当前人数
- status: 课程状态（报名中/进行中/已结束）
- create_time: 创建时间
- update_time: 更新时间

#### 培训报名表 (training_enrollment)
- enrollment_id: 报名ID（主键）
- course_id: 课程ID
- emp_id: 员工ID
- enrollment_date: 报名日期
- status: 报名状态（待审核/已通过/已拒绝）
- attendance: 签到状态
- score: 培训成绩
- create_time: 创建时间
- update_time: 更新时间
