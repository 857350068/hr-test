# 人力资源数据中心 - 启动说明

## 🚀 快速启动

### 方式一:使用启动脚本(推荐)

1. **双击运行 `start.bat`**
2. **选择启动方式:**
   - 输入 `1`: 只启动后端
   - 输入 `2`: 只启动前端
   - 输入 `3`: 同时启动前后端(推荐)
   - 输入 `4`: 退出

3. **等待服务启动完成**
   - 后端启动成功后会显示: `人力资源数据中心启动成功!`
   - 前端启动成功后会显示: `http://localhost:3000`

4. **访问系统**
   - 前端地址: http://localhost:3000
   - 后端地址: http://localhost:8080/api

5. **测试登录**
   - 用户名: `admin` / 密码: `123456`
   - 用户名: `hr001` / 密码: `123456`

### 方式二:手动启动

#### 启动后端

```bash
# 进入后端目录
cd backend

# 打包项目(首次运行)
mvn clean package -DskipTests

# 启动后端
java -jar target/hr-data-center-1.0.0.jar
```

#### 启动前端

```bash
# 进入前端目录
cd frontend

# 安装依赖(首次运行)
npm install

# 启动前端
npm run dev
```

## 📋 前置要求

### 必需软件
- ✅ JDK 1.8 或更高版本
- ✅ Node.js 16 或更高版本
- ✅ MySQL 8.0 或更高版本
- ✅ Maven 3.6 或更高版本

### 检查环境

```bash
# 检查Java版本
java -version

# 检查Node.js版本
node -v

# 检查Maven版本
mvn -v

# 检查MySQL服务
mysql -u root -p
```

## 🗄️ 数据库初始化

### 方法一:自动初始化(推荐)

启动脚本会自动检查数据库,如果不存在会提示您执行初始化脚本。

### 方法二:手动初始化

```bash
# 登录MySQL
mysql -u root -p

# 执行初始化脚本
source backend/src/main/resources/sql/init.sql
```

## 🛑 停止服务

### 方法一:使用停止脚本

双击运行 `stop.bat`

### 方法二:手动停止

```bash
# 停止后端(Windows)
taskkill /F /IM java.exe

# 停止前端(Windows)
taskkill /F /IM node.exe

# 停止后端(Linux/Mac)
kill -9 $(jps -l | grep hr-data-center | awk '{print $1}')

# 停止前端(Linux/Mac)
kill -9 $(ps aux | grep 'vite' | grep -v grep | awk '{print $2}')
```

## 🐛 常见问题

### 问题1:编译失败

**错误信息:**
```
[ERROR] COMPILATION ERROR
```

**解决方案:**
```bash
# 清理并重新编译
cd backend
mvn clean package -DskipTests
```

### 问题2:数据库连接失败

**错误信息:**
```
Communications link failure
```

**解决方案:**
1. 检查MySQL服务是否启动
2. 检查数据库配置文件: `backend/src/main/resources/application.yml`
3. 确保数据库已创建: `hr_datacenter`
4. 执行初始化脚本

### 问题3:端口被占用

**错误信息:**
```
Port 8080 was already in use
```

**解决方案:**
```bash
# 查找占用端口的进程
netstat -ano | findstr :8080

# 结束进程
taskkill /PID <进程ID> /F

# 或者修改配置文件中的端口号
# 编辑 backend/src/main/resources/application.yml
# 修改 server.port 为其他端口
```

### 问题4:前端依赖安装失败

**错误信息:**
```
npm ERR! code ECONNREFUSED
```

**解决方案:**
```bash
# 使用淘宝镜像
npm config set registry https://registry.npmmirror.com

# 重新安装
cd frontend
npm install
```

### 问题5:前端启动失败

**错误信息:**
```
Error: Cannot find module 'xxx'
```

**解决方案:**
```bash
# 删除node_modules和package-lock.json
cd frontend
rm -rf node_modules package-lock.json

# 重新安装
npm install

# 重新启动
npm run dev
```

## 📊 功能测试

### 测试登录功能

1. 访问 http://localhost:3000
2. 输入用户名: `admin`
3. 输入密码: `123456`
4. 点击登录按钮
5. 登录成功后跳转到首页

### 测试员工管理功能

1. 登录成功后,点击左侧菜单"员工管理"
2. 查看员工列表
3. 点击"新增员工"按钮
4. 填写员工信息并保存
5. 点击"编辑"按钮修改员工信息
6. 点击"删除"按钮删除员工

### 测试API接口

使用Postman或curl测试接口:

```bash
# 登录接口
curl -X POST http://localhost:8080/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username":"admin","password":"123456"}'

# 获取员工列表
curl -X GET "http://localhost:8080/api/employee/list?page=1&size=10"

# 获取员工总数
curl -X GET http://localhost:8080/api/employee/total
```
