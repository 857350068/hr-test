# 前端UI重新设计任务清单

## 阶段一：配色系统统一

### 任务1.1：更新CSS变量文件
- **文件**：`frontend-legacy/src/assets/styles/variables.css`
- **操作**：更新CSS变量定义，添加完整的配色系统
- **验收标准**：
  - 包含主色系（#409EFF蓝色）
  - 包含功能色系（绿、橙、红）
  - 包含中性色系（灰阶）
  - 包含背景色系
  - 包含阴影和尺寸变量

### 任务1.2：检查并替换硬编码颜色
- **文件**：所有`.vue`和`.css`文件
- **操作**：查找所有硬编码的颜色值，替换为CSS变量
- **验收标准**：
  - 无硬编码颜色值（如#409EFF、#67C23A等）
  - 所有颜色使用var()引用

### 任务1.3：删除紫色渐变代码
- **文件**：所有`.vue`和`.css`文件
- **操作**：查找并删除所有紫色、indigo、violet色系及渐变
- **验收标准**：
  - 无紫色系颜色（#8B5CF6, #6366F1, #A855F7, #7C3AED）
  - 无indigo系颜色（#4F46E5, #4338CA）
  - 无紫蓝渐变

## 阶段五：质量检查

### 任务5.1：视觉检查
- **操作**：检查所有页面，确保符合设计规范
- **验收标准**：
  - 无紫色渐变
  - 无indigo色系
  - 配色符合60-30-10法则
  - 背景为纯色或细微纹理
  - 所有颜色使用CSS变量
  - 悬停效果流畅
  - 响应式布局正常

### 任务5.2：性能检查
- **操作**：检查CSS文件大小和加载性能
- **验收标准**：
  - CSS文件大小 < 100KB
  - 无重复样式定义
  - 过渡动画流畅
  - 首屏渲染 < 2秒

- **操作**：测试所有页面功能，确保无破坏
- **验收标准**：
  - 登录功能正常
  - 仪表盘数据展示正常
  - 侧边栏导航正常
  - 响应式布局正常
