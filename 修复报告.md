### 1. 必须修复的问题（致命错误）

#### ❌ 问题：Leave实体类表名映射错误
- **文件位置**：`backend/src/main/java/com/hr/datacenter/entity/Leave.java:16`
- **问题描述**：
  - Java代码中：`@TableName("leave")`
  - MySQL数据库中：表名是 `emp_leave`
  - 影响：程序运行时会报错 `Table 'leave' doesn't exist`

#### ✅ 修复方案
```java
// 修改前
@Data
@TableName("`leave`")  // 错误
public class Leave implements Serializable {

// 修改后
@Data
@TableName("emp_leave")  // 正确
public class Leave implements Serializable {
```

#### ✅ 修复结果
- Leave实体类表名已正确映射到 `emp_leave`
- 程序不再报表不存在的错误
- 所有请假相关功能可以正常使用
