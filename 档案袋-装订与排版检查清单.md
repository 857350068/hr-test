# 毕业设计（论文）档案袋检查清单（按你提供要求整理）

## 一、完整档案袋三大部分

- [ ] 1）毕业设计（论文）装订册
- [ ] 2）毕业设计（论文）附件材料
- [ ] 3）毕业设计（论文）教师评审手册

---

## 二、版面格式检查（A4）

- [ ] 纸张：A4
- [ ] 上边距：3.0 cm
- [ ] 下边距：2.5 cm
- [ ] 左边距：2.5 cm
- [ ] 右边距：2.0 cm
- [ ] 页脚：1.75 cm
- [ ] 封面与封底：白色铜版纸（120g-150g）

---

## 三、装订册组成与检查项

### 1）封面

- [ ] 题目不超过 20 个字
- [ ] 题目简练准确（可分两行）
- [ ] 学生、专业、学号、指导教师、系部等信息齐全

### 2）正文前置部分

- [ ] 英文题目与作者
- [ ] 任务书（教师填写并签字生效）
- [ ] 开题报告
- [ ] 原创性声明与版权授权书
- [ ] 中文摘要（300-500字）
- [ ] 英文摘要（与中文内容一致）
- [ ] 关键词（3-5个，通用技术词）
- [ ] 目录（三级标题、与正文一致）

### 3）正文主体

- [ ] 绪论（研究目的、意义、国内外综述、研究内容）
- [ ] 主体（方案比较、设计过程、原理分析、参数/数据对比）
- [ ] 结论（单独成章，不超过两页为宜）
- [ ] 本科字数不少于 8000 字（或不少于20页）

### 4）正文后置部分

- [ ] 参考文献不少于 20 篇
- [ ] 致谢
- [ ] 附录
- [ ] 实验数据表、图纸（超 3# 图幅单独装订）

### 5）封底

- [ ] 完整装订

---

## 四、附件材料组成与检查项

### 1）封面与目录

- [ ] 附件封面
- [ ] 附件目录（按二级标题）

### 2）英文文献翻译

- [ ] 1.1 英文原文（不少于3000词）
- [ ] 1.2 中文翻译（完整对应）

### 3）专业阅读书目

- [ ] 不少于 10 篇
- [ ] 每篇不少于 300 字
- [ ] 在开题报告或正文中有体现
