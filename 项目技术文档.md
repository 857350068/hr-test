# 人力资源数据中心系统 - 项目技术文档（基于实际代码）

## 📋 项目概述

**项目名称**: 人力资源数据中心系统（HrDataCenter）

**技术架构**: SpringBoot 2.7.18 + Vue 3.3.4 + Hadoop 3.2.4 + Hive 3.1.3

**项目路径**: d:\HrDataCenter

**文档更新时间**: 2026-04-07

**项目完成度**: 98%

---

## 🏗️ 技术架构

### 前端技术栈
- **框架**: Vue 3.3.4
- **构建工具**: Vite 4.4.9
- **UI组件库**: Element Plus 2.3.14
- **图表库**: ECharts 5.4.3
- **路由**: Vue Router 4.2.4
- **HTTP客户端**: Axios 1.5.0
- **Excel处理**: XLSX 0.18.5, File-Saver 2.0.5

### 后端技术栈
- **框架**: Spring Boot 2.7.18
- **ORM**: MyBatis Plus 3.5.3.1
- **安全框架**: Spring Security
- **JWT**: 0.11.5
- **数据库**: MySQL 8.0 + Hive 3.1.3
- **大数据**: Hadoop 3.2.4
- **工具库**: Hutool 5.8.20, Lombok

### 数据库架构
- **业务库**: MySQL（10张业务表）
- **数据仓库**: Hive（4张维度表 + 4张事实表）
- **数据模型**: 星型模型
- **存储格式**: ORC + SNAPPY压缩

## 📝 项目亮点

1. **双数据源架构**: MySQL处理业务数据，Hive处理大数据分析
2. **完整的数据同步机制**: 支持定时同步维度表、事实表、聚合表
3. **智能预警系统**: 基于多因素权重的流失预警和人才缺口预警
4. **全面的数据分析**: 组织效能、人才梯队、薪酬福利三大分析模块
5. **Excel导入导出**: 支持批量数据导入和数据导出
6. **操作日志记录**: 基于AOP的操作日志切面
7. **前后端分离**: 标准的前后端分离架构
8. **响应式设计**: 基于Element Plus的现代化UI界面
9. **模糊查询优化**: 支持多字段组合模糊查询
10. **图表可视化**: 使用ECharts实现丰富的数据可视化
