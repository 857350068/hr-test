## 🚀 启动方式

### 方式一:一键启动(推荐)

1. **双击运行 `start.bat`**
2. **选择 `3`**(同时启动前后端)
3. **等待启动完成**
4. **访问 http://localhost:3000**
5. **使用测试账号登录**:
   - 用户名: `admin` / 密码: `123456`
   - 用户名: `hr001` / 密码: `123456`

### 方式二:手动启动

#### 启动后端

```bash
cd hr-backend
java -jar target/hr-data-center-1.0.0.jar
```

#### 启动前端

```bash
cd hr-frontend
npm run dev
```

## 📞 使用说明

### 测试账号

| 用户名 | 密码 | 角色 | 说明 |
|-------|------|------|------|
| admin | 123456 | 系统管理员 | 拥有所有权限 |
| hr001 | 123456 | HR专员 | 普通HR员工 |

### 访问地址

- **前端地址**: http://localhost:3000
- **后端地址**: http://localhost:8080/api
- **数据库**: localhost:3306/hr_datacenter

### API接口

#### 认证接口
- `POST /api/auth/login` - 用户登录
- `POST /api/auth/logout` - 用户登出

#### 员工接口
- `GET /api/employee/list` - 获取员工列表
- `GET /api/employee/{id}` - 获取员工详情
- `POST /api/employee/add` - 新增员工
- `PUT /api/employee/update` - 更新员工
- `DELETE /api/employee/delete/{id}` - 删除员工
- `GET /api/employee/total` - 获取员工总数
